# todo.txt Changelog

## [Initial Release] - {PR_MERGE_DATE}

- Search, filter, sort, and group tasks from your todo.txt file
- Create tasks with priority, project, context, and due date via form or raw todo.txt syntax
- Mark tasks complete and optionally archive them to done.txt
- Mark completed tasks as incomplete (when not using archive mode)
- Browse and restore completed tasks from done.txt
- Menu bar item showing pending task count with overdue indicator
- Shared cache for instant menu bar updates after any mutation
- Full todo.txt format support: priorities, creation dates, completion dates, projects, contexts, due dates, and arbitrary key:value tags
